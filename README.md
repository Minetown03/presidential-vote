# HRS Presidential Vote

A simple, secure voting system for the HRS Presidential Election.

## Features

- Dark, modern interface
- Three candidate options (NiKn, CRIs, sassogamer)
- One vote per device
- Secure vote counting
- Admin panel for vote monitoring
- Responsive design for all devices

## How to Use

1. Share the main URL with voters
2. Voters can cast their vote by clicking on their preferred candidate
3. Each device can only vote once
4. Access the admin panel by adding `?admin=hrs2024` to the URL

## Technical Details

- Built with HTML, JavaScript, and Tailwind CSS
- Uses localStorage for vote storage
- No server-side components required
- Easy to deploy and share

## Admin Access

To access the admin panel:
1. Add `?admin=hrs2024` to the URL
2. View vote counts
3. Reset votes if needed
